运行时 API 示例 ​

本页面展示了 VitePress 提供的一些运行时 API 的使用方法。

主要的 useData() API 可以用来访问当前页面的站点、主题和页面数据。它在 .md 和 .vue 文件中都可以使用：

md
<script setup>
import { useData } from 'vitepress'

const { theme, page, frontmatter } = useData()
</script>

## 结果

### 主题数据

<pre>{{ theme }}</pre>

### 页面数据

<pre>{{ page }}</pre>

### 页面前置数据

<pre>{{ frontmatter }}</pre>
